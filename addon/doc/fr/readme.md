## Description
**Dictionary Display Lite** est votre nettoyeur de parole personnel pour les boîtes de dialogue de dictionnaire de NVDA. Vous connaissez cette annonce interminable à chaque fois que vous parcourez une liste d'entrées ?

> Motif : OBS ; Remplacer par : O B S ; casse : désactivé ; Type : Expression régulière

Avec **Dictionary Display Lite**, vous n'entendrez que l'essentiel – instantanément :

> OBS O B S désactivé Expression régulière

Plus de « Motif : », « Remplacer par : », « casse : » ou « Type : » – seulement les valeurs, prononcées fluidement d'un seul trait. Il fonctionne silencieusement en arrière-plan, rendant la navigation dans votre dictionnaire plus rapide et bien plus agréable.

> [!TIP]
> **Zéro configuration requise :** Installez l'extension, ouvrez n'importe quel dictionnaire (Par défaut, Voix ou Temporaire) et placez le focus sur un élément de la liste. NVDA prononcera automatiquement la version condensée. C'est tout !

## Pourquoi vous allez l'adorer
Si vous gérez de nombreuses entrées de dictionnaire, vous savez à quel point les annonces complètes peuvent être répétitives. Vos oreilles se fatiguent et votre cerveau travaille davantage pour filtrer le bruit. **Dictionary Display Lite** s'occupe du filtrage pour vous, ne laissant que les informations dont vous avez réellement besoin.

## Ce qu'il supprime
* **« Motif : »** – vous savez déjà que c'est un motif.
* **« Remplacer par : »** – la deuxième partie est évidemment le remplacement.
* **« casse : »** – vous entendrez « activé » ou « désactivé » juste après le remplacement.
* **« Type : »** – le dernier mot vous indique le type de règle (Expression régulière, Caractère générique, etc.).
* Tous les points-virgules et espaces superflus – pour un flux fluide et continu.

## Gardez toujours le contrôle total
Ne vous inquiétez pas – le texte original détaillé n'est pas perdu. Il est seulement masqué de la parole. Si vous avez besoin de réentendre toutes les étiquettes, vous pouvez :
* Appuyer sur **NVDA+Tab** pour lire l'objet actuel avec tous les détails, ou
* Utiliser les commandes de revue d'objet (**NVDA+pavé numérique**) pour explorer le texte brut.
